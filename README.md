# Super-Mario-World-in-Python
Данный проект является моей попыткой перенести базовые вещи из игры "Super Mario World" в Python. Сразу говорю, не ждите какого-нибудь шедевра или нормальной игры, так как я очень часто забрасывал его.
# Особенности
1. Попытка создать физику из оригинальной "Super Mario World".
2. Доступный исходный код.
3. Перенос базовых механик.
# Инструменты
1. Python
2. VsCode
3. библиотека PyGame.
